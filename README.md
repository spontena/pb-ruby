# pb-ruby

Pandorabots API module for Ruby. Please read the [documentation](http://developer.pandorabots.com/docs).

## Installation

Add this line to your application's Gemfile:

    gem 'pandorabots_api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pandorabots_api
